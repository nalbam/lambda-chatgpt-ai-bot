# 사용법 예시 및 실제 시나리오

## 🚀 실제 사용 시나리오

### 📝 단순 대화 및 질문 답변

**채널에서 멘션:**
```
@botname 안녕하세요! 어떤 일을 도와드릴까요?
@botname 양자 컴퓨팅을 쉽게 설명해주세요
@botname Python에서 리스트를 정렬하는 방법은?
```

**다이렉트 메시지:**
```
머신러닝과 딥러닝의 차이점은 뭔가요?
자바스크립트 함수 문법 알려주세요
오늘 날씨는 어때요? (일반적인 대화)
```

### 🎨 이미지 생성 요청

**기본 이미지 생성:**
```
@botname 귀여운 고양이 그려줘
@botname 미래 도시의 스카이라인을 그려주세요
@botname Draw a robot in cyberpunk style
@botname 회사 로고 디자인해줘 - 심플하고 모던한 스타일로
```

**상세한 프롬프트:**
```
@botname 석양을 배경으로 한 바다가 보이는 카페에서 커피를 마시는 사람 그려줘
@botname 3D 렌더링 스타일로 우주정거장 내부를 그려주세요
```

### 🖼️ 이미지 분석 요청

**이미지 업로드 + 분석 요청:**
```
[이미지 업로드] 
@botname 이 이미지에서 무엇을 볼 수 있나요?
@botname 이 차트의 데이터를 분석해주세요
@botname 이 코드 스크린샷을 설명해주세요
@botname 이 다이어그램의 의미는 무엇인가요?
```

### 📋 스레드 요약 기능

**스레드 내에서 요약 요청:**
```
@botname 스레드 요약해줘
@botname 이 토론 내용을 정리해주세요
@botname summarize this thread
@botname 회의 내용을 요약해서 정리해줘
```

**스레드 요약 결과 예시:**
```
📋 스레드 요약 (15개 메시지)

주요 주제: 새로운 프로젝트 기획 논의

핵심 내용:
- 팀원들이 React vs Vue.js 기술 스택에 대해 논의
- 프로젝트 일정은 3개월로 합의
- 디자인 시스템 구축이 우선순위로 결정

중요한 결정사항:
- React를 메인 프레임워크로 선택
- 디자인 시스템 작업을 1개월 내 완료
- 주간 회의는 매주 화요일 오전 10시로 확정

참여자 의견:
- 김개발: React 생태계가 더 안정적이라는 의견
- 이디자인: 컴포넌트 재사용성 강조
- 박기획: 일정 준수의 중요성 언급
```

## 🔥 복합 작업 요청 (고급 기능)

### 설명 + 이미지 생성 조합

**AI 설명 + 시각화:**
```
@botname AI에 대해 설명하고 로봇 이미지도 그려줘
@botname 머신러닝 알고리즘을 요약하고 관련 다이어그램도 만들어줘
@botname 블록체인 기술을 설명하고 블록체인 구조 이미지도 생성해줘
```

**실행 과정:**
1. 🤖 요청 분석 중...
2. 📋 2개 작업을 처리합니다... (예상 시간: 25초)
3. ⚙️ 작업 1/2: AI 기술 설명 처리 중...
4. 💭 [AI에 대한 상세한 설명이 스트리밍으로 전송]
5. ⚙️ 작업 2/2: 로봇 이미지 생성 처리 중...
6. 🎨 [생성된 로봇 이미지가 업로드]
7. ✅ 모든 작업이 완료되었습니다.

### 분석 + 생성 조합

**이미지 분석 + 개선안 제시:**
```
[차트 이미지 업로드]
@botname 이 차트를 분석하고 개선 방안도 써줘

[코드 스크린샷 업로드]  
@botname 이 코드를 설명하고 최적화 버전도 작성해줘

[UI 디자인 이미지 업로드]
@botname 이 디자인을 분석하고 개선된 버전 이미지도 만들어줘
```

### 다단계 복합 작업

**개념 설명 + 코드 + 시각화:**
```
@botname 퀵소트 알고리즘 설명해주고, Python 코드 예시도 주고, 동작 과정 다이어그램도 그려줘
```

**실행 결과:**
1. 퀵소트 알고리즘에 대한 상세한 설명 (스트리밍)
2. Python 퀵소트 구현 코드 (스트리밍)  
3. 퀵소트 동작 과정을 보여주는 시각적 다이어그램 (이미지)

## 💡 효과적인 사용 팁

### 명확한 요청 작성법

**좋은 예시:**
```
@botname Python 딕셔너리 자료구조를 초보자도 이해할 수 있게 설명하고, 실제 사용 예시 코드도 보여줘
@botname 회사 브랜딩용 로고를 심플하고 모던한 스타일로 디자인해줘 - 파란색과 흰색 조합으로
```

**피해야 할 예시:**
```
@botname 이거 해줘 (구체적이지 않음)
@botname 그냥 그림 하나 그려줘 (세부사항 부족)
```

### 스레드 활용법

**긴 대화는 스레드 사용:**
```
채널 메시지: @botname 프로젝트 계획 도와줘
└── 스레드 1: 기술 스택 추천해줘
└── 스레드 2: 일정 계획 세워줘  
└── 스레드 3: 스레드 요약해줘
```

### 이미지 업로드 최적화

**지원 포맷:** PNG, JPG, JPEG, GIF
**권장 크기:** 10MB 이하
**최적 해상도:** 1024x1024 이하

## 🔧 문제 해결

### 응답이 없는 경우
1. 멘션이 정확한지 확인 (@botname)
2. 잠시 후 다시 시도
3. 다이렉트 메시지로 테스트

### 이미지 생성 실패
1. 프롬프트를 더 구체적으로 작성
2. 부적절한 내용이 포함되지 않았는지 확인
3. 영어로 다시 시도

### 복합 작업 중단
1. 개별 작업으로 나누어 요청
2. 네트워크 상태 확인
3. 잠시 후 재시도
